# BLV is my life

```
├─ Configs/                        # Бекапы конфигов 
    ├─ printer.txt
    ├─ BLV.3mf                     # Конфигурация проектом (если не работает `.orca_printer`)
    └─ BLV.orca_printer            # Последняя рабочая конфигурация
├─ Fan/
    ├─ BLV - 5015.stl              # Крепление улитки
    ├─ BLV- AirBlade.stl           # Направляющая обдува
    └─ BLV- FanStand.stl           # Крепление обдува
├─ BLV - BackPlate.stl             # Передняя панель
├─ BLV - BackTiltHolder.stl        # Передняя панель
├─ BLV - CarriageBody.stl          # Каретка
├─ BLV - FanBLTouch.stl            # Крепление вентилятора и тача
├─ BLV - WireDown.stl              # Направляющая для косы
└─ BLV - WireUp.stl                # Крышка для направляющей косы

```
